Tag Experiments

Using the <article> Tag

Figuring out how to use the different tags relating to chunks of written text is a challenge. So for today's coding challenge, I'm going to try and use both the <article> and the <aside> tags correctly (and a bunch of others, as you'll notice if you check the code for this page).


Why Use the <aside> Tag?

The <aside> tag allows you to ellobrate on information pertaining to the surrounding text without having to interrupt the flow of reading. So readers may just skip over the <aside> section and come back later. Although, to be fair, that will only fully work once I know how to style this part with CSS. But I'll get there. In addition, the <aside> tag is useful for people using assitive technology, such as screen readers. The tag indicates that the text it encloses is a side note, definiton etc., so the sudden change of subject does not end up leading to confusion.


Article or Section?

I still find it challenging to figure out whether I should use the <article> tag or the <section> tag in any given context. I chose the <article> one for this text based on the information I found in this post on Geeks for Geeks. But I may be wrong. My guess is this will become easier the more I code and have to make that call. As a professionl language nerd, these nuances really matter to me. So knowing that they matter in coding as well makes my language nerd heart smile.
